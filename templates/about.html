Job Application Tracker

A web app that helps job seekers organize and monitor their job applications in one place - tracking status and follow-ups so nothing falls through the cracks.

Project type

CRUD app with external API integration. Users create, read, update, and delete job application records. The app enriches records with live company data fetched from an external API.

Endpoints implemented

MethodRouteDescription
GET/Home page - dashboard if logged in
GET/applicationsList all job applications for the logged-in user
GET/POST/applications/newCreate a new job application record
GET/applications/<id>Show detail view for one job application
GET/POST/applications/<id>/editUpdate an existing job application
POST/applications/<id>/deleteDelete a job application
GET/login/githubInitiate GitHub OAuth flow
GET/auth/github/callbackHandle GitHub OAuth callback

Web-based data source

  • Primary API: Clearbit Company API
  • Endpoint: https://company.clearbit.com/v2/companies/find?domain=DOMAIN
  • What it provides: Company name, description, industry, employee count, and logo
  • Fallback API: API Ninjas Logo API — used when Clearbit returns no result
  • Manual fallback: User can enter company details manually if both APIs return no result
